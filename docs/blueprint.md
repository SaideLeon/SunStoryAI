# **App Name**: Hello Sunrise

## Core Features:

- Basic Display: Display 'Hello world' on the screen.
- User Customization: Allow the user to change the displayed text.
- Personalized Greeting: AI powered personalized greeting depending on the time of day. Uses time as a tool to give an appropriate greeting.

## Style Guidelines:

- Paleta quase monocromática: #0c0c0c base, superfícies em #141414 e #1c1c1c
- Sem âmbar agressivo — substituído por #c9b99a (âmbar desbotado, mais sofisticado)
- Bordas em rgba ultrafinas em vez de border-slate-700 opaco
- Grid fixo 1fr + 360px — painel direito com largura definida, sem float
- Sem cards com rounded-2xl e shadow-xl por todo lado — superfícies planas com borda fina
- Espaço negativo generoso nos padding
- Clean, centered layout with ample whitespace for readability.
- Subtle fade-in animation for the greeting text.
